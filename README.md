## 소상공인을 위한 Smart Warehouse 🏭
> 소상공인을 위한 ‘ All-In-One 스마트 창고 ’
- 쉽고 간단한 웹 인터페이스와 RFID를 통한 실시간 입출고 시스템
- 창고와 고객주문 건을 효율적으로 관리할 수 있고, 다품종 소량 생산에 최적화된 자동 출고 Machine을 적용
